Create an Autonomy app for news fact-checking with this workflow:
1. Editor pastes article
2. Agent extracts claims
3. Parallel agents fact-check each claim via Brave Search MCP
4. Report agent synthesizes findings

First, create a README with an architecture diagram. Then deploy and test in Autonomy.
