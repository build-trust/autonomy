# A guide for coding agents to select and configure models

**Models** are the large language models that **Agents** rely on to understand, reason, and make autonomous decisions.

> **Key Terms**: See the [definitions section of the main guide](https://autonomy.computer/docs/_for-coding-agents.md#definitions).

*Keywords: claude, llama, model selection, claude-sonnet, deepseek*

This guide covers available models and helps you select the right model for an agent.

## Supported Models

Autonomy Computer supports a wide range of models from different providers. When creating agents, you can specify which model to use with the `Model()` constructor.

### Claude Models

* `claude-3-7-sonnet-v1` - Advanced reasoning capabilities
* `claude-opus-4-v1` - Most capable Claude model for complex tasks
* `claude-sonnet-4-v1` - **Recommended** - Latest generation Claude model with optimal balance

### Llama Models

* `llama3.2` - Meta's Llama 3.2 model
* `llama3.3` - Meta's Llama 3.3 model
* `llama4-maverick` - Llama 4 Maverick variant
* `llama4-scout` - Llama 4 Scout variant

### AWS Nova Models

* `nova-lite-v1` - AWS Nova Lite model
* `nova-micro-v1` - AWS Nova Micro model
* `nova-pro-v1` - AWS Nova Pro model
* `nova-premier-v1` - AWS Nova Premier model

### Other Models

* `deepseek-r1` - DeepSeek's reasoning model

## Example of how to use a model

```python
from autonomy import Agent, Model, Node

async def main(node):
    # Using Claude Sonnet 4 (recommended for most applications)
    await Agent.start(
        node=node,
        name="assistant",
        instructions="You are a helpful assistant",
        model=Model("claude-sonnet-4-v1")
    )

    # Using a different model for a different use case
    await Agent.start(
        node=node,
        name="reasoning-agent",
        instructions="You are a reasoning specialist",
        model=Model("claude-opus-4-v1")
    )

Node.start(main)
```

### Recommendations for selecting a model

* **For most applications**: `claude-sonnet-4-v1` (recommended default)
* **For fast, simple tasks**: `llama4-maverick`
* **For complex reasoning**: `claude-opus-4-v1` or `deepseek-r1`
* **For cost-effective solutions**: `llama4-maverick` or `nova-micro-v1`
* **For maximum capability**: `claude-opus-4-v1` or `nova-premier-v1`

All models support the same interface, so you can easily switch between them by changing the model name in your `Model()` constructor.
